### 在macOS安装

##### 安装*VirtualBox*虚拟机软件

因为minikube创建K8S虚拟机是通过Virtualbox来做的（当然还有其它driver，比如KVM，vmware等等）

##### 安装*minikube*

Minikube是一个快速搭建单节点Kubenetes集群的工具

```
curl -LO https://storage.googleapis.com/minikube/releases/latest/minikube-darwin-amd64
```

##### 运行minikube程序创建k8s

```
 minikube start #去创建k8s环境
 minikube stop #关闭k8s环境
 minikube dashboard #启动ui界面
```

##### 安装kubectl

```
yuyudeMacBook-Pro:Desktop yuyu$ curl -LO "https://dl.k8s.io/release/$(curl -L -s https://dl.k8s.io/release/stable.txt)/bin/darwin/amd64/kubectl"

yuyudeMacBook-Pro:Desktop yuyu$ mv kubectl /usr/local/bin/

yuyudeMacBook-Pro:Desktop yuyu$ sudo chomd +x /usr/local/bin/kubectl 

yuyudeMacBook-Pro:Desktop yuyu$ kubectl version
```
