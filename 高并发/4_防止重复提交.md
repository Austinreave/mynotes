##### 防止重复提交：

1. js层面，在用户点击提交订单后禁用按钮。

2. 使用Redis incrde 原子性递增计数

   ```
   //针对同一单生成固定唯一标识
   orderNum=memberId+productId;
   
   //条数加1
   count = redis.increment(orderNum, 1);
   if (count == 1) {
   	//设置有效期2秒
     redis.expire(orderNum, 2, TimeUnit.SECONDS);
   }
   
   if (count > 1) {
      resultMap.put("retCode", "500");     
      resultMap.put("retMsg", "请不要重复提交订单");
      return resultMap;
   }
   ```
